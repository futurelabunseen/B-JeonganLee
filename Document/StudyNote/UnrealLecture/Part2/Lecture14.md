# 14강: 게임플로우 다듬기

- 강의 목표
  - 게임을 이루는 전체 구성 요소의 파악하고 이를 보강
  - 전체 게임 플로우가 잘 동작하도록 캐릭터 스택과 아이템 데이터를 다듬기
- 강의 과제
  - 게임에서 사용할 다양한 아이템 정보 및 관련 데이터를 효과적으로 변경하고 적용할 방법을 정리하시오.

## 게임 플로우를 위해 보강할 내용

- 죽었을 때의 NPC와 플레이어의 처리
- 이동 속도의 적용
- 포션/스크롤 아이템의 추가 구현과 캐릭터의 적용
- 스탯 기능 및 기능의 보강

## 정리

- 죽는 상태에 대한 플레이어와 NPC의 설정
- 전체 게임 구성 요소의 이해
- 캐릭터 데이터 조정과 다양한 아이템 설정 및 구현
